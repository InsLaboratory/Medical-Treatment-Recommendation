# Offline Reinforcement Learning for Sepsis Treatment Recommendation

This repository contains the full pipeline for an offline deep reinforcement
learning project applied to sepsis treatment optimization in the ICU setting.
The project uses the Health Gym synthetic dataset (Kuo et al., 2022) derived
from MIMIC-III and targets the training and evaluation of conservative offline
RL algorithms against the historical clinician policy.

## Scientific Context

Sepsis is a life-threatening condition requiring rapid and complex treatment
decisions in the ICU. This project frames treatment recommendation as a
Markov Decision Process (MDP) and applies offline RL to learn policies
from retrospective clinical data, without any online environment interaction.

**Algorithms compared:**
- Behavior Cloning (BC) — supervised imitation baseline
- Conservative Q-Learning (CQL) — penalises out-of-distribution actions
- Batch-Constrained Q-Learning (BCQ) — constrains the policy to the data support

**Evaluation:** Off-policy evaluation (OPE) via Weighted Importance Sampling
(WIS) and Fitted Q-Evaluation (FQE), benchmarked against the clinician policy.

---

## Dataset

The dataset is the Health Gym Sepsis synthetic dataset (v1.0.0) published by
Kuo et al. (2022) in *Scientific Data*. It contains 43,280 ICU timesteps
(4-hour intervals) and is synthetically generated from MIMIC-III.

**Access:** The dataset is available on PhysioNet after signing a data use agreement.  
Link: https://physionet.org/content/health-gym/1.0.0/

Once downloaded, place the csv file in the `data/` directory.

## MDP Specification

| Component | Definition |
|-----------|-----------|
| State S | 56-dimensional normalised physiological vector [0, 1] |
| Action A | 25 discrete actions (5 fluid levels x 5 vasopressor levels) |
| Reward R | +15 / -15 terminal (survival/readmission) + intermediate -lambda * delta_SOFA |
| Horizon | Variable-length episodes (mean ~8 steps, 4h each) |
| Discount | gamma = 0.99 |
